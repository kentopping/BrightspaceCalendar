on linux run:
1. sudo apt install chromium-driver

on windows replace the chromedriver with a windows chromedriver 
1. check the verison of chrome you have by clicking the 3 dot drop down under the X button, hover over help and click about chrome
2. go to https://chromedriver.chromium.org/downloads, click the version of chrome you have and download the windows zip
2. unzip and replace the chromedriver in the BrightspaceCalendar folder
